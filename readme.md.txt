# Aprende a Leer

Sitio interactivo para que niños aprendan a leer mediante imágenes y opciones. 

## Características

- Imagen + 2 opciones
- Pronunciación de la palabra al ver la imagen y al elegir opción
- Efectos de acierto para motivar
- Totalmente responsivo (optimizado para móviles)

## Uso

1. Abre `index.html` en tu navegador.
2. Sube este proyecto a GitHub y actívalo con GitHub Pages si quieres publicarlo.

¡Disfrútalo!
